지원 형식: .pcap, .cap, .pcapng (모든 PCAP 변형 지원)

🔍 검색 문법 도움말

기본 검색:
  • 192.168.1.1 - IP 주소로 검색
  • TCP - 프로토콜로 검색
  • 80 - 포트 번호로 검색
필드별 검색:
  • src:192.168.1.1 - 소스 IP
  • dst:192.168.1.100 - 목적지 IP
  • protocol:TCP - 특정 프로토콜
  • port:80 - 특정 포트
  • size:>1000 - 크기 조건 (>, <, =)
조합 검색:
  • src:192.168.1.1 AND dst:192.168.1.100
  • protocol:TCP OR protocol:UDP
  • port:80 OR port:443
